Basic coding...

Learn some basic coding to help you in your job...

Front-end

  • HTML: is the basic mark-up langauage creates the basic document that the browser reads to know how to render the page
  • CSS: is the language we use to style our HTML, we can add colours, size, fonts, animations
  • JavaScript: is the langauage we use to add interactions to our site

Back-end

Examples of back-end languages that essentially do the same thing...

  • Java (not to be confused with JavaScript)
  • PHP (Lightweight and easy-to-use but relatively slow)
  • C# (Used in the Windows stack)
  • Python (Good for big data processing)
  • JavaScript (we can now use JS on the back-end with NodeJS)

Databases

SQL: (relational databases)

  • Oracle (the one we use)
  • MySQL (open-source and works well across the board)
  • SQLServer (Used in the Windows stack, works well with C#)

NoSQL: (non-relational databases)

  • MongoDB (My personal favourite – works well with JavaScript)
  • Cassandra (more enterprise-friendly)
